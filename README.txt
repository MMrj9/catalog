
How to run:
	-First should initialize the vagrant machine using "vagrant up" and then access it via "vagrant ssh".
	-Then you should run the app by running the following command "python app.py."
	-You should now be able to access the app at localhost:5000

You can now check all the categories and products that exist. You can login using google auth and then you can also create/edit/delete your own categories and products.
